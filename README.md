# 山音溪

山音溪博客网站前端项目，此项目基于 [Ant Design Pro](https://pro.ant.design)

[后端传送门][blog-back-end]

### 安装 `node_modules`

`npm install` or `yarn`

### 运行 project

```bash
npm start
```

### 打包 project

```bash
npm run build
```

### 检查

```bash
npm run lint
```

### 使用脚本自动修复某些 lint 错误

```bash
npm run lint:fix
```

### 测试

```bash
npm test
```

[blog-back-end]: https://github.com/Koyamane/blog-back-end
